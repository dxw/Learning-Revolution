Feature: Venue admin

In order to keep the venue data clean
As an admin
I should be able to be able to add, edit and delete venues

Scenario: Adding a valid venue

  1. When I go to the venue admin index page
  2. And I follow "New venue"
  3. And I fill in "Name" with "Venue name"
  4. And I fill in "Address 1" with "Address 1 name"
  5. And I fill in "Address 2" with "Address 2 name"
  6. And I fill in "Address 3" with "Address 3 name"
  7. And I fill in "City" with "City name"
  8. And I fill in "County" with "County name"
  9. And I fill in "Postcode" with "Postcode name"
  10. And I press "Create"
  11. Then I should see "Venue created successfully"
  12. And I should see "Venue name"
  13. And I am on the venue admin index page

Scenario: Submitting an empty form

  1. When I go to the venue admin index page
  2. And I follow "New venue"
  3. And I press "Create"
  4. Then I should see "errors prohibited this venue from being saved"
  5. And I should see "Name can't be blank"
  6. And I should see "Postcode can't be blank"
  7. And I am on the venue admin index page

Scenario: Editing a valid venue

  1. Given a valid venue called "Venue 1"
  2. When I go to the venue admin index page
  3. And I follow "Venue 1"
  4. And I fill in "Name" with "New Venue name"
  5. And I press "Save"
  6. Then I should see "Venue saved successfully"
  7. And I should see "New Venue name"
  8. And I am on the venue admin index page

Scenario: Removing the new venue in a duplicate

  1. Given a valid venue called "venue 1"
  2. And a valid venue called "venue 2"
  3. When I go to the venue duplicates page
  4. Then I should see "venue 1"
  5. And I should see "venue 2"
  6. When I press "Remove new venue"
  7. Then I should see "venue 2 was deleted"
  8. And I should not see "venue 1"

Scenario: Removing the original venue in a duplicate

  1. Given a valid venue called "venue 1"
  2. And a valid venue called "venue 2"
  3. When I go to the venue duplicates page
  4. Then I should see "venue 1"
  5. And I should see "venue 2"
  6. When I press "Remove original venue"
  7. Then I should see "venue 1 was deleted"
  8. And I should not see "venue 2"

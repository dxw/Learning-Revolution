Feature: Event admin

In order to keep the event data clean
As an admin
I should be able to be able to add, edit and delete events

Scenario: Adding a valid event

  1. When I go to the event admin index page
  2. And I follow "New event"
  3. And I fill in "Title" with "event title"
  4. And I fill in "Description" with "event description"
  5. And I fill in "Theme" with "event theme"
  6. And I fill in "Type" with "event type"
  7. And I fill in "Stage" with "event stage"
  8. And I select "November 23, 2004 11:20" as the "Start" date and time
  9. And I select "November 23, 2004 12:20" as the "End" date and time
  10. And I fill in "Cost" with "£1.50"
  11. And I fill in "Min Age" with "13"
  12. And I fill in "Venue" with "London"
  13. And I fill in "Organisation" with "BIS"
  14. And I fill in "Contact Name" with "event organiser"
  15. And I fill in "Contact Phone Number" with "020 8547 3847"
  16. And I fill in "Contact Email Address" with "contact@test.com"
  17. And I fill in "Further Information" with "There isn't really any further info"
  18. And I fill in "Additional Notes" with "Additional Notes"
  19. And I check "Published"
  20. And I fill in "Picture" with "http://www.google.co.uk/intl/en_uk/images/logo.gif"
  21. And I check "Featured"
  22. And I press "Create"
  23. Then I should see "Event created successfully"
  24. And I should see "event title"
  25. And I am on the event admin index page

Scenario: Submitting an empty form

  1. When I go to the event admin index page
  2. And I follow "New event"
  3. And I press "Create"
  4. Then I should see "error prohibited this event from being saved"
  5. And I should see "Title can't be blank"
  6. And I am on the event admin index page

Scenario: Editing a valid event

  1. Given a valid event called "event 1"
  2. When I go to the event admin index page
  3. And I follow "event 1"
  4. And I fill in "Title" with "New event title"
  5. And I press "Save"
  6. Then I should see "Event saved successfully"
  7. And I should see "New event title"
  8. And I am on the event admin index page

Scenario: Removing the new event in a duplicate

  1. Given a valid event called "event 1"
  2. And a valid event called "event 2"
  3. When I go to the event duplicates page
  4. Then I should see "event 1"
  5. And I should see "event 2"
  6. When I press "Remove new event"
  7. Then I should see "event 2 was deleted"
  8. And I should not see "event 1"

Scenario: Removing the original event in a duplicate

  1. Given a valid event called "event 1"
  2. And a valid event called "event 2"
  3. When I go to the event duplicates page
  4. Then I should see "event 1"
  5. And I should see "event 2"
  6. When I press "Remove original event"
  7. Then I should see "event 1 was deleted"
  8. And I should not see "event 2"

Scenario: Editing an event mid moderation

  1. Given a valid event called "event 1"
  2. And a valid event called "event 2"
  3. When I go to the event duplicates page
  4. And I follow "Edit original event"
  5. And I press "Save"
  6. Then I should be on the event duplicates page

Scenario: Moderating events with approval

  1. Given a valid event called "event 1"
  2. And a valid event called "event 2"
  3. When I go to the events moderation page
  4. Then I should see "event 1"
  5. When I press "Approve"
  6. Then I should see "event 1 has been published"
  7. And I should see "event 2"

Scenario: Moderating events with deletion

  1. Given a valid event called "event 1"
  2. And a valid event called "event 2"
  3. When I go to the events moderation page
  4. Then I should see "event 1"
  5. When I press "Delete"
  6. Then I should see "event 1 has been deleted"
  7. And I should see "event 2"

Scenario: Moderating events with skipping

  1. Given a valid event called "event 1"
  2. And a valid event called "event 2"
  3. When I go to the events moderation page
  4. Then I should see "event 1"
  5. When I follow "Skip"
  6. Then I should not see "event 1 has been deleted"
  7. And I should not see "event 1 has been published"
  8. And I should see "event 2"
